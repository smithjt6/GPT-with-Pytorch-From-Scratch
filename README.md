# GPT-with-Pytorch-From-Scratch
A simple and spelled out repository for building a GPT! 

This repo is a simple, throughly commented and referenced implementation of a GPT (think like OpenAI GPT-2/3). Starting at the tokenizer level and ending at the training, validation and generation of new data!
We will be using Tiny Stories which is available at huggingface datasets (https://huggingface.co/datasets/roneneldan/TinyStories)
